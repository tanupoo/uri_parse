uri_parse
=========

*uri_parse* parses several types of URI, and return the following dict.

    {
        'url_scheme' : '<url_scheme>',
          'url_host' : '<url_host>',
          'url_path' : '<ulr_path>',
              'host' : '<ip address or host name>',
              'port' : '<port number or service name>'
    }
